Tentang aku

Aku adalah seorang pembelajar yang sedang menjelajahi dunia pemrograman dari nol. Saat ini, aku fokus mempelajari HTML, CSS, dan JavaScript untuk membangun website interaktif, sekaligus belajar Python untuk memahami logika pemrograman yang lebih luas. HTML mengajarkanku bagaimana membentuk kerangka sebuah website, CSS membuatnya indah dan rapi, JavaScript memberinya nyawa, dan Python membuka jalan menuju analisis data, otomasi, hingga kecerdasan buatan. Perjalanan ini bukan sekadar mempelajari kode, tapi juga belajar berpikir logis, kreatif, dan tidak mudah menyerah. Setiap baris kode yang kutulis adalah langkah kecil menuju mimpiku menjadi seorang developer yang handal.
